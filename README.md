# 💀 LUST DAR

Advanced Termux Security & Network Tool

---

## ABOUT

LUST DAR is a lightweight CLI tool designed for Termux.  
It provides system information, network utilities, and basic diagnostic tools in a simple terminal interface.

---

## FEATURES

- System Information (CPU, RAM, Kernel, Storage)
- Network Information
- Public IP Checker
- DNS Lookup
- Ping Test
- File Hash Check (SHA256)
- Process Viewer
